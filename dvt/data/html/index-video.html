Distant Viewing Toolkit — Video Visualization

This is the default visualization of extracted metadata using the distant viewing toolkit. The objective of the visualization is to demonstrate how computer vision can be used as a way of seeing the visual material. Below are selected frames from the input video file. The first frame shows detected objects and people; the second frame illustrates movement (hue is direction and saturation is speed) using optical flow. If sound sources were included, the next two images show the tone and spectrogram. Other extracted fields are shown on the right. See the distant viewing project page for further information.
